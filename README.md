## ğŸ“– Materi Praktikum

### Modul 1: Object Orientation
Pengenalan konsep dasar pemrograman berorientasi objek, meliputi:
- Definisi class dan object
- Atribut dan method
- Constructor
- Static members

### Modul 2: Enkapsulasi dan Relasi Objek
Implementasi enkapsulasi dan berbagai jenis relasi antar objek:
- Access modifiers (private, public, protected)
- Getter dan Setter
- Relasi objek (komposisi, agregasi, asosiasi)

### Modul 3: Exception dan Assertion
Penanganan error dan validasi program:
- Try-catch-finally blocks
- Throw dan throws
- Custom exceptions
- Assertion untuk debugging

### Modul 4: Inheritance
Konsep pewarisan dalam OOP:
- Superclass dan subclass
- Method overriding
- Super keyword
- Constructor chaining

### Modul 5: Abstract Class dan Interface
Abstraksi dan kontrak dalam pemrograman:
- Abstract class dan abstract method
- Interface dan implementasinya
- Multiple inheritance melalui interface
- Default methods

### Modul 6: Polimorfisme dan Binding
Polimorfisme dalam berbagai bentuknya:
- Inclusion polymorphism
- Coercion polymorphism
- Dynamic binding
- Static binding

### Modul 7: Generics
Pemrograman generik untuk fleksibilitas tipe data:
- Generic class dan method
- Type parameters
- Bounded type parameters
- Wildcards

### Modul 8: Koleksi
Framework koleksi Java:
- List, Set, Map
- ArrayList, LinkedList
- HashMap, TreeMap
- Iterator dan collections operations

### Modul 9: Persistensi
Penyimpanan dan pembacaan data:
- File I/O
- Serialization
- Database connectivity
- CRUD operations

### Modul 10: Modularisasi
Pengembangan aplikasi modular:
- Java modules
- GUI programming dengan Swing/JavaFX
- JDBC untuk koneksi database
- MVC pattern

### Modul 11: Lintas Paradigma (Lambda)
Pemrograman fungsional dalam Java:
- Lambda expressions
- Functional interfaces
- Stream API
- Method references

### Responsi Pra UTS
Review dan latihan soal untuk persiapan ujian tengah semester.
